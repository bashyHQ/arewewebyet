En- & Decoding

Encoding is one of those things you don't really want to bother about, but just rely on a well tested library for – there are just too many edge and quirky cases. Luckily, as a system language, this is one of the things rust shines at – and offers a set of nice packages already.

  • data-encoding

    Efficient and customizable data-encoding functions

  • encoding

    Character encoding support for Rust

  • base64

    encodes and decodes base64 as bytes or utf8

  • urlencoded

    Decoding middleware for url-encoded data. For use with Iron.

  • multipart

    A backend-agnostic extension for HTTP libraries that provides support for POST multipart/form-data requests on both client and server.

  • bitsparrow

    BitSparrow implementation in Rust
